# Forge

A personal knowledge system built on Claude Code.

Forge can help you turn any assignment/article into a list of skills/concepts you need to learn. 
Forge will then go concept by concept asking you to state your understanding to identify your gaps.
Forge then enters what you learn into a flashcard that becomes part of a daily flashcard system to help you retain what you learned.

## Context

The current repository is preloaded with the concepts extracted from Accounting and Corporate Finance final exams. I am testing the skill now to launch Forge as an On-The-Job training project over the summer.

## Setup

Give Claude a link to this repository (github.com/rberg27/ozian-forge-skill) and ask Claude to install the folders for you.

Open this folder in Claude Code and say:

> "install skills"

## Usage

Just talk to Claude:

> "Start studying accounting"
> "forge quiz"
> "forge review"
